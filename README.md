# chemical-property-predictor-ml

## Overview 
This project is based on machine learning and determines whether a chemical solution is acidic, basic or neutral taking into account its pH value.

## Features
- pH of chemical solution is taken as input.
- utilizes a machine learning model for classification.

- ## technologies used
- python
- Scikit-learn (Decision Tree Classifier)

  ## How it works
  The machine learning model used learns from the data of ph values and their categories ( acid,base, neutral). Then, it processes the new input pH and identifies the chemical nature of solution provided accordingly.
